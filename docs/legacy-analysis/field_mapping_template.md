# Шаблон сопоставления полей импорта

## Назначение
Этот документ нужен для поэтапного разбора legacy-парсинга Excel/MXL и фиксации соответствия между:
- исходными колонками файла;
- промежуточными значениями парсера;
- новой доменной моделью;
- правилами валидации;
- спорными местами и эвристиками.

Документ заполняется отдельно по каждому типу файла или шаблону.
